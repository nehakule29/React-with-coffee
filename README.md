# React Learning Projects

A collection of React projects built while learning from the **Chai aur Code** YouTube channel by Hitesh Choudhary.

## Projects

- **Password Generator** - Generate secure passwords with customizable options
- **Currency Converter** - Real-time currency conversion with live exchange rates
-  **Basic projects** - Background Changer, etc. (The initial projects for understanding of hooks and other stuff)

## Tech Stack

React • React Router • React Hooks • Tailwind CSS • Vite

## Getting Started

```bash
# Clone the repository
git clone https://github.com/yourusername/react-learning-projects.git

# Install dependencies
npm install

# Run development server
npm run dev
```
